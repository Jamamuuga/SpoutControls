GET STARTED

1) Run : SPOUTCONTROLLER\CINDER\binaries\CinderSpoutSenderControl.exe
   This is a Spout Sender application like any other but with control functions included

2) Run : SPOUTCONTROLLER\SpoutController.exe
   This is a Spout controller program that will control the Cinder sender which is running
   Try the controls.

3) Run : SPOUTRECEIVER\SpoutReceiver.exe
   This is a Spout receiver and will recieve the output of the sender.
   Try the controls and the changed sender will be received.

4) Open "quickstart.txt" to review this information
   Then read the manual...
